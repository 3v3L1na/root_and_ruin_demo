# Root and Ruin — Interactive Narrative Demo
A branching, choice-driven dark gothic fantasy prototype built using Twine (SugarCube 2).

👉 **[CLICK HERE TO PLAY THE DEMO LIVE IN YOUR BROWSER](https://3v3l1na.github.io/root_and_ruin_demo/)**

## Technical Design Features:
* **Systemic State Tracking:** Developed a backend framework to track player stamina (`$iva_stamina`) and character alignment (`$konstantin_respect`).
* **Conditional Story Architecture:** Implemented variable-dependent dialogue gates (`$guilt_level`) that dynamically shift narrative outcomes based on player psychology.
* **Data & Narrative Integration:** Positioned at the intersection of complex narrative worldbuilding and programmatic logic, utilizing principles from my broader data analytics portfolio.
